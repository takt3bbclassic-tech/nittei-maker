# 日程候補メーカー（調整さん・伝助 自動作成）

自然文（例：「6月の平日18時開始」「月水金は19時から、火曜は21時から、木曜はなし、土日は13時から」）から
候補日リストを生成し、調整さん / 伝助のイベントページを自動作成して共有リンクを表示するWebアプリ。

## 構成

- `index.html` … フロントエンド（パーサー・UI、スマホ対応）
- `api/create.js` … Vercel Serverless Function。調整さん（CSRFトークン取得→POST）/ 伝助（直接POST）でイベント作成しURLを返す

## デプロイ（Vercel）

```bash
npm i -g vercel   # 初回のみ
cd nittei-maker
vercel            # 案内に従う（無料枠でOK）
vercel --prod
```

GitHubリポジトリに置いて vercel.com からインポートでも可。設定は不要（ビルドコマンドなし、`api/` は自動認識）。

## 注意

- 自動作成は両サービスとも**非公式**（フォーム送信の自動化）。仕様変更で動かなくなる可能性あり。
  その場合もアプリの「候補をコピー」→公式作成ページに貼り付け、で利用可能。
  伝助は自動作成失敗時、入力済みの確認画面を自動で開くフォールバック付き。
- 祝日データは2026〜2027年分を内蔵。2028年以降を使う場合は `HOLIDAYS`（index.html内）に追記。
- 調整さんのイベントは一定期間アクセスがないと自動削除される（公式仕様）。
